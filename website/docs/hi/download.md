## डाउनलोड

नवीनतम वर्जन के लिए GitHub Releases पेज पर जाएं:

👉 [GitHub Releases](https://github.com/AQBot-Desktop/AQBot/releases)

### macOS

- **Apple Silicon (M1/M2/M3/M4)**: `AQBot_x.x.x_aarch64.dmg` डाउनलोड करें
- **Intel**: `AQBot_x.x.x_x64.dmg` डाउनलोड करें

### Windows

- **इंस्टॉलर संस्करण**: `AQBot_x.x.x_x64-setup.exe` डाउनलोड करें
- **पोर्टेबल संस्करण (Portable)**: `AQBot_x.x.x_windows-x64-portable.zip` डाउनलोड करें — एक्सट्रैक्ट करें और उपयोग करें, इंस्टॉलेशन की आवश्यकता नहीं

### Linux

- **Debian/Ubuntu**: `.deb` पैकेज डाउनलोड करें
- **अन्य वितरण**: `.AppImage` डाउनलोड करें

## इंस्टॉलेशन

### macOS

1. डाउनलोड किया `.dmg` फ़ाइल खोलें
2. AQBot को Applications फोल्डर में ड्रैग करें
3. पहली बार लॉन्च पर, System Settings → Privacy & Security में इसे अनुमति दें

### Windows

**इंस्टॉलर संस्करण：**
1. डाउनलोड किया इंस्टॉलर चलाएं
2. इंस्टॉलेशन पूरी करने के लिए विज़ार्ड का पालन करें
3. Start Menu या डेस्कटॉप शॉर्टकट से लॉन्च करें

**पोर्टेबल संस्करण (Portable)：**
1. `.zip` फ़ाइल को किसी भी फोल्डर में एक्सट्रैक्ट करें
2. `AQBot.exe` पर डबल-क्लिक करके चलाएं

### Linux

```bash
# Debian/Ubuntu
sudo dpkg -i AQBot_x.x.x_amd64.deb

# AppImage
chmod +x AQBot_x.x.x_amd64.AppImage
./AQBot_x.x.x_amd64.AppImage
```
