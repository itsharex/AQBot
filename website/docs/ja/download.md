## システム要件

| プラットフォーム | 最低バージョン |
|----------------|--------------|
| macOS | 11.0 (Big Sur) 以降 |
| Windows | 10 (1803) 以降 |
| Linux | 主要ディストリビューション |

## ダウンロード

最新バージョンは GitHub Releases ページからダウンロードしてください：

👉 [GitHub Releases](https://github.com/AQBot-Desktop/AQBot/releases)

### macOS

- **Apple Silicon (M1/M2/M3/M4)**: `AQBot_x.x.x_aarch64.dmg` をダウンロード
- **Intel**: `AQBot_x.x.x_x64.dmg` をダウンロード

### Windows

- **インストーラー版**: `AQBot_x.x.x_x64-setup.exe` をダウンロード
- **ポータブル版 (Portable)**: `AQBot_x.x.x_windows-x64-portable.zip` をダウンロード — 解凍するだけですぐ使える、インストール不要

### Linux

- **Debian/Ubuntu**: `.deb` パッケージをダウンロード
- **その他のディストリビューション**: `.AppImage` をダウンロード

## インストール

### macOS

1. ダウンロードした `.dmg` ファイルを開く
2. AQBot をアプリケーションフォルダにドラッグ
3. 初回起動時は、システム設定 → プライバシーとセキュリティで許可する

### Windows

**インストーラー版：**
1. ダウンロードしたインストーラーを実行
2. ウィザードに従ってインストールを完了
3. スタートメニューまたはデスクトップのショートカットから起動

**ポータブル版 (Portable)：**
1. `.zip` ファイルを任意のフォルダに解凍
2. `AQBot.exe` をダブルクリックして起動

### Linux

```bash
# Debian/Ubuntu
sudo dpkg -i AQBot_x.x.x_amd64.deb

# AppImage
chmod +x AQBot_x.x.x_amd64.AppImage
./AQBot_x.x.x_amd64.AppImage
```
