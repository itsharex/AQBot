## تنزيل

قم بزيارة صفحة GitHub Releases للإصدار الأحدث:

👉 [GitHub Releases](https://github.com/AQBot-Desktop/AQBot/releases)

### macOS

- **Apple Silicon (M1/M2/M3/M4)**: نزّل `AQBot_x.x.x_aarch64.dmg`
- **Intel**: نزّل `AQBot_x.x.x_x64.dmg`

### Windows

- **نسخة المثبّت**: نزّل `AQBot_x.x.x_x64-setup.exe`
- **النسخة المحمولة (Portable)**: نزّل `AQBot_x.x.x_windows-x64-portable.zip` — فك الضغط واستخدم مباشرة، بدون تثبيت

### Linux

- **Debian/Ubuntu**: نزّل حزمة `.deb`
- **توزيعات أخرى**: نزّل `.AppImage`

## التثبيت

### macOS

1. افتح ملف `.dmg` المحمّل
2. اسحب AQBot إلى مجلد Applications
3. عند الإطلاق الأول، اسمح به في إعدادات النظام ← الخصوصية والأمان

### Windows

**نسخة المثبّت：**
1. شغّل المثبّت المحمّل
2. اتبع المعالج لإكمال التثبيت
3. أطلق من قائمة ابدأ أو اختصار سطح المكتب

**النسخة المحمولة (Portable)：**
1. فك ضغط ملف `.zip` في أي مجلد
2. انقر مزدوجاً على `AQBot.exe` لتشغيله

### Linux

```bash
# Debian/Ubuntu
sudo dpkg -i AQBot_x.x.x_amd64.deb

# AppImage
chmod +x AQBot_x.x.x_amd64.AppImage
./AQBot_x.x.x_amd64.AppImage
```
