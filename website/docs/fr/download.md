## Configuration requise

| Plateforme | Version minimale |
|-----------|----------------|
| macOS | 11.0 (Big Sur) ou supérieur |
| Windows | 10 (1803) ou supérieur |
| Linux | Principales distributions |

## Téléchargement

Visitez la page GitHub Releases pour la dernière version :

👉 [GitHub Releases](https://github.com/AQBot-Desktop/AQBot/releases)

### macOS

- **Apple Silicon (M1/M2/M3/M4)** : Téléchargez `AQBot_x.x.x_aarch64.dmg`
- **Intel** : Téléchargez `AQBot_x.x.x_x64.dmg`

### Windows

- **Version installable** : Téléchargez `AQBot_x.x.x_x64-setup.exe`
- **Version portable (Portable)** : Téléchargez `AQBot_x.x.x_windows-x64-portable.zip` — Extraire et utiliser, aucune installation nécessaire

### Linux

- **Debian/Ubuntu** : Téléchargez le paquet `.deb`
- **Autres distributions** : Téléchargez `.AppImage`

## Installation

### macOS

1. Ouvrez le fichier `.dmg` téléchargé
2. Faites glisser AQBot dans le dossier Applications
3. Au premier lancement, autorisez-le dans Réglages Système → Confidentialité et sécurité

### Windows

**Version installable :**
1. Exécutez l'installateur téléchargé
2. Suivez l'assistant pour terminer l'installation
3. Lancez depuis le menu Démarrer ou le raccourci bureau

**Version portable (Portable) :**
1. Extrayez le fichier `.zip` dans n'importe quel dossier
2. Double-cliquez sur `AQBot.exe` pour lancer

### Linux

```bash
# Debian/Ubuntu
sudo dpkg -i AQBot_x.x.x_amd64.deb

# AppImage
chmod +x AQBot_x.x.x_amd64.AppImage
./AQBot_x.x.x_amd64.AppImage
```
