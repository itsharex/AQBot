# 下載安裝

<DownloadHero />

## 系統需求

| 平台 | 最低版本 |
|------|---------|
| macOS | 11.0 (Big Sur) 及以上 |
| Windows | 10 (1803) 及以上 |
| Linux | 各主流發行版 |

## 下載

前往 GitHub Releases 頁面下載最新版本：

👉 [GitHub Releases](https://github.com/AQBot-Desktop/AQBot/releases)

### macOS

- **Apple Silicon (M1/M2/M3/M4)**: 下載 `AQBot_x.x.x_aarch64.dmg`
- **Intel**: 下載 `AQBot_x.x.x_x64.dmg`

### Windows

- **安裝版**: 下載 `AQBot_x.x.x_x64-setup.exe`
- **綠色免安裝版 (Portable)**: 下載 `AQBot_x.x.x_windows-x64-portable.zip` — 解壓即用，無需安裝

### Linux

- **Debian/Ubuntu**: 下載 `.deb` 套件
- **其他發行版**: 下載 `.AppImage`

## 安裝

### macOS

1. 開啟下載的 `.dmg` 檔案
2. 將 AQBot 拖入「應用程式」資料夾
3. 首次執行時，在「系統設定 → 隱私權與安全性」中允許執行

### Windows

**安裝版：**
1. 執行下載的安裝程式
2. 依精靈完成安裝
3. 從開始功能表或桌面捷徑啟動

**綠色免安裝版 (Portable)：**
1. 解壓 `.zip` 檔案到任意目錄
2. 雙擊 `AQBot.exe` 即可執行

### Linux

```bash
# Debian/Ubuntu
sudo dpkg -i AQBot_x.x.x_amd64.deb

# AppImage
chmod +x AQBot_x.x.x_amd64.AppImage
./AQBot_x.x.x_amd64.AppImage
```
