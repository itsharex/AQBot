## Системные требования

| Платформа | Минимальная версия |
|----------|------------------|
| macOS | 11.0 (Big Sur) или новее |
| Windows | 10 (1803) или новее |
| Linux | Основные дистрибутивы |

## Скачать

Посетите страницу GitHub Releases для последней версии:

👉 [GitHub Releases](https://github.com/AQBot-Desktop/AQBot/releases)

### macOS

- **Apple Silicon (M1/M2/M3/M4)**: Скачайте `AQBot_x.x.x_aarch64.dmg`
- **Intel**: Скачайте `AQBot_x.x.x_x64.dmg`

### Windows

- **Установщик**: Скачайте `AQBot_x.x.x_x64-setup.exe`
- **Портативная версия (Portable)**: Скачайте `AQBot_x.x.x_windows-x64-portable.zip` — Распаковать и использовать, установка не требуется

### Linux

- **Debian/Ubuntu**: Скачайте пакет `.deb`
- **Другие дистрибутивы**: Скачайте `.AppImage`

## Установка

### macOS

1. Откройте скачанный файл `.dmg`
2. Перетащите AQBot в папку «Программы»
3. При первом запуске разрешите его в Системных настройках → Конфиденциальность и безопасность

### Windows

**Установщик：**
1. Запустите скачанный установщик
2. Следуйте мастеру для завершения установки
3. Запустите из меню «Пуск» или ярлыка на рабочем столе

**Портативная версия (Portable)：**
1. Распакуйте `.zip` файл в любую папку
2. Дважды щёлкните `AQBot.exe` для запуска

### Linux

```bash
# Debian/Ubuntu
sudo dpkg -i AQBot_x.x.x_amd64.deb

# AppImage
chmod +x AQBot_x.x.x_amd64.AppImage
./AQBot_x.x.x_amd64.AppImage
```
