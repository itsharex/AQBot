## System Requirements

| Platform | Minimum Version |
|----------|----------------|
| macOS | 11.0 (Big Sur) or later |
| Windows | 10 (1803) or later |
| Linux | Major distributions |

## Download

Visit the GitHub Releases page for the latest version:

👉 [GitHub Releases](https://github.com/AQBot-Desktop/AQBot/releases)

### macOS

- **Apple Silicon (M1/M2/M3/M4)**: Download `AQBot_x.x.x_aarch64.dmg`
- **Intel**: Download `AQBot_x.x.x_x64.dmg`

### Windows

- **安装版**: Download `AQBot_x.x.x_x64-setup.exe`
- **绿色免安装版 (Portable)**: Download `AQBot_x.x.x_windows-x64-portable.zip` — 解压即用，无需安装

### Linux

- **Debian/Ubuntu**: Download the `.deb` package
- **Other distros**: Download the `.AppImage`

## Installation

### macOS

1. Open the downloaded `.dmg` file
2. Drag AQBot to the Applications folder
3. On first launch, allow it in System Settings → Privacy & Security

### Windows

**安装版：**
1. Run the downloaded installer
2. Follow the wizard to complete installation
3. Launch from Start Menu or desktop shortcut

**绿色免安装版 (Portable)：**
1. 解压 `.zip` 文件到任意目录
2. 双击 `AQBot.exe` 即可运行

### Linux

```bash
# Debian/Ubuntu
sudo dpkg -i AQBot_x.x.x_amd64.deb

# AppImage
chmod +x AQBot_x.x.x_amd64.AppImage
./AQBot_x.x.x_amd64.AppImage
```
