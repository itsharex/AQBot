## Requisitos del sistema

| Plataforma | Versión mínima |
|-----------|----------------|
| macOS | 11.0 (Big Sur) o posterior |
| Windows | 10 (1803) o posterior |
| Linux | Distribuciones principales |

## Descarga

Visita la página de GitHub Releases para la última versión:

👉 [GitHub Releases](https://github.com/AQBot-Desktop/AQBot/releases)

### macOS

- **Apple Silicon (M1/M2/M3/M4)**: Descarga `AQBot_x.x.x_aarch64.dmg`
- **Intel**: Descarga `AQBot_x.x.x_x64.dmg`

### Windows

- **Versión instalable**: Descarga `AQBot_x.x.x_x64-setup.exe`
- **Versión portable (Portable)**: Descarga `AQBot_x.x.x_windows-x64-portable.zip` — Descomprime y usa, sin necesidad de instalación

### Linux

- **Debian/Ubuntu**: Descarga el paquete `.deb`
- **Otras distribuciones**: Descarga `.AppImage`

## Instalación

### macOS

1. Abre el archivo `.dmg` descargado
2. Arrastra AQBot a la carpeta Aplicaciones
3. En el primer inicio, permítelo en Configuración del Sistema → Privacidad y Seguridad

### Windows

**Versión instalable：**
1. Ejecuta el instalador descargado
2. Sigue el asistente para completar la instalación
3. Inicia desde el menú Inicio o el acceso directo del escritorio

**Versión portable (Portable)：**
1. Descomprime el archivo `.zip` en cualquier carpeta
2. Haz doble clic en `AQBot.exe` para ejecutar

### Linux

```bash
# Debian/Ubuntu
sudo dpkg -i AQBot_x.x.x_amd64.deb

# AppImage
chmod +x AQBot_x.x.x_amd64.AppImage
./AQBot_x.x.x_amd64.AppImage
```
