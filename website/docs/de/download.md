## Systemanforderungen

| Plattform | Mindestversion |
|-----------|---------------|
| macOS | 11.0 (Big Sur) oder neuer |
| Windows | 10 (1803) oder neuer |
| Linux | Wichtige Distributionen |

## Download

Besuchen Sie die GitHub Releases-Seite für die neueste Version:

👉 [GitHub Releases](https://github.com/AQBot-Desktop/AQBot/releases)

### macOS

- **Apple Silicon (M1/M2/M3/M4)**: Laden Sie `AQBot_x.x.x_aarch64.dmg` herunter
- **Intel**: Laden Sie `AQBot_x.x.x_x64.dmg` herunter

### Windows

- **Installer-Version**: Laden Sie `AQBot_x.x.x_x64-setup.exe` herunter
- **Portable Version (Portable)**: Laden Sie `AQBot_x.x.x_windows-x64-portable.zip` herunter — Entpacken und direkt verwenden, keine Installation nötig

### Linux

- **Debian/Ubuntu**: Laden Sie das `.deb`-Paket herunter
- **Andere Distributionen**: Laden Sie `.AppImage` herunter

## Installation

### macOS

1. Öffnen Sie die heruntergeladene `.dmg`-Datei
2. Ziehen Sie AQBot in den Ordner Programme
3. Beim ersten Start erlauben Sie es unter Systemeinstellungen → Datenschutz & Sicherheit

### Windows

**Installer-Version：**
1. Führen Sie den heruntergeladenen Installer aus
2. Folgen Sie dem Assistenten, um die Installation abzuschließen
3. Starten Sie über das Startmenü oder die Desktop-Verknüpfung

**Portable Version (Portable)：**
1. Entpacken Sie die `.zip`-Datei in ein beliebiges Verzeichnis
2. Doppelklicken Sie auf `AQBot.exe` zum Starten

### Linux

```bash
# Debian/Ubuntu
sudo dpkg -i AQBot_x.x.x_amd64.deb

# AppImage
chmod +x AQBot_x.x.x_amd64.AppImage
./AQBot_x.x.x_amd64.AppImage
```
