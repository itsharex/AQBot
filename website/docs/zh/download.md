## 系统要求

| 平台 | 最低版本 |
|------|---------|
| macOS | 11.0 (Big Sur) 及以上 |
| Windows | 10 (1803) 及以上 |
| Linux | 各主流发行版 |

## 下载

前往 GitHub Releases 页面下载最新版本：

👉 [GitHub Releases](https://github.com/AQBot-Desktop/AQBot/releases)

### macOS

- **Apple Silicon (M1/M2/M3/M4)**: 下载 `AQBot_x.x.x_aarch64.dmg`
- **Intel**: 下载 `AQBot_x.x.x_x64.dmg`

### Windows

- **安装版**: 下载 `AQBot_x.x.x_x64-setup.exe`
- **绿色免安装版 (Portable)**: 下载 `AQBot_x.x.x_windows-x64-portable.zip` — 解压即用，无需安装

### Linux

- **Debian/Ubuntu**: 下载 `.deb` 包
- **其他发行版**: 下载 `.AppImage`

## 安装

### macOS

1. 打开下载的 `.dmg` 文件
2. 将 AQBot 拖入「应用程序」文件夹
3. 首次运行时，在「系统设置 → 隐私与安全性」中允许运行

### Windows

**安装版：**
1. 运行下载的安装程序
2. 按向导完成安装
3. 从开始菜单或桌面快捷方式启动

**绿色免安装版 (Portable)：**
1. 解压 `.zip` 文件到任意目录
2. 双击 `AQBot.exe` 即可运行

### Linux

```bash
# Debian/Ubuntu
sudo dpkg -i AQBot_x.x.x_amd64.deb

# AppImage
chmod +x AQBot_x.x.x_amd64.AppImage
./AQBot_x.x.x_amd64.AppImage
```
