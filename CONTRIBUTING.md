本文件是 AQBot 项目的贡献者协议（Contributor License Agreement, CLA）。你向本仓库提交 Pull Request、commit、patch、issue 附件中的代码片段，或以其他方式提交可合并到项目中的内容，即表示你已阅读、理解并同意本协议。

本协议适用于你提交给本项目的所有贡献内容，包括但不限于：

- 源代码、脚本、配置、测试、构建文件；
- 文档、示例、注释、翻译；
- UI 资源、图标、图片、设计稿、数据文件；
- 通过 Pull Request、commit、patch、issue、讨论区或其他 GitHub 功能提交并明确供本项目采用的内容。

仅用于问题描述、运行日志、截图或复现说明，且未明确要求合并进项目的内容，不视为本协议下的贡献内容。
